# 🤖 Assistente Virtual Inteligente para WhatsApp

## 📌 Sobre o Projeto

Assistente Virtual Inteligente desenvolvido em Python para automatizar o atendimento no WhatsApp Web utilizando Selenium, Chromium e Inteligência Artificial.

O sistema monitora mensagens recebidas, interpreta as solicitações dos usuários através de IA e gera respostas automáticas em tempo real, proporcionando um atendimento rápido, inteligente e escalável.

---

## 🚀 Funcionalidades

* Abertura automática do WhatsApp Web
* Monitoramento de mensagens recebidas
* Respostas automáticas inteligentes
* Integração com Inteligência Artificial
* Atendimento automatizado em tempo real
* Reconhecimento de perguntas e contexto
* Gerenciamento automático de sessão
* Funcionamento através do Chromium e Selenium
* Estrutura escalável para novos recursos

---

## 🧠 Inteligência Artificial

O projeto utiliza Inteligência Artificial para compreender mensagens recebidas e gerar respostas contextualizadas automaticamente, permitindo interações mais naturais e eficientes com os usuários.

---

## 🛠 Tecnologias Utilizadas

* Python
* Selenium
* Chromium
* Inteligência Artificial
* WebDriver Manager
* Git
* GitHub
* Linux Ubuntu

---

## ▶️ Execução

```bash
python whatsapp.py
```

Após a inicialização, o sistema abre automaticamente o WhatsApp Web e inicia o monitoramento das conversas para responder mensagens utilizando IA.
